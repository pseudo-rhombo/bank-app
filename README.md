# Banking Default Prediction App

## Description
This web app predicts whether a bank customer is likely to **default on a loan** based on their personal and financial details. Users can input information such as age, income, years employed, loan amount, credit history, and account balance.  

The app uses a **pre-trained Decision Tree model** to provide real-time predictions, helping banks or financial institutions make data-driven decisions efficiently.

## Features
- Simple and intuitive interface for entering customer data  
- Real-time loan default prediction  
- Clear output showing likelihood of default  

## Use Cases
- Assess risk for loan applicants  
- Demonstrate machine learning applications in finance
